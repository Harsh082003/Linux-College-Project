# System Maintenance Toolkit (Advanced)

A Bash-based command-line toolkit for basic Linux system administration tasks.

## Features
- System Information (CPU, Memory, Disk)
- File Organizer (categorize files into folders)
- Backup & Restore (tar.gz)
- Log Analyzer (journalctl or syslog)
- Process Manager (view top processes and kill by PID)
- Disk Cleanup (package cleanup + large file listing)

## Usage
1. Make the script executable:
bash
chmod +x toolkit.sh

2. Run:
bash
./toolkit.sh
